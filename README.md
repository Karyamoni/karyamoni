# Karyamoni

Karyamoni is a virtual clothing try-on application for e-commerce stores, starting as an IKAS application and later expanding to Shopify and other popular CMS builders.

The product vision is simple: help shoppers understand fit before they buy, so store owners can reduce return rates caused by inconsistent sizing across brands, cuts, and product categories.

## Product Concept

Karyamoni will let IKAS merchants add a 3D virtual fitting cabin to their stores with a simple app-style installation. Shoppers should use the try-on experience directly inside the merchant's product page flow, without being redirected to a separate Karyamoni website.

The first market is Turkey, so Turkish will be treated as a primary language from day one. English will also be supported to prepare the product for international growth.

## Initial Focus

- Build as an IKAS application first.
- Make merchant integration feel close to one-click.
- Support Shopify after the IKAS experience is validated.
- Create an embedded 3D fitting cabin experience for clothing try-on.
- Reduce sizing-related returns for e-commerce owners.
- Provide Turkish and English product experiences.
- Design for easy CMS integration and merchant onboarding.
